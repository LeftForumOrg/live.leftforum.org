INTRODUCTION
------------

Marinelli is a 3 column, tableless layout theme with a wide image banner and a
"top-tabbed" primary-links system.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/project/marinelli

 * Additional documentation for this project can be found at:
   https://www.drupal.org/node/1000274

 * To submit bug reports and feature suggestions, or to track changes:
   https://www.drupal.org/project/issues/marinelli

The images included with the theme were taken by Jason Flatt during his trip
through Mexico and Central America. The license for them is covered under the
license for this project, GPLv2+.


REQUIREMENTS
------------

This theme has the following requirements:

 * hoverIntent (http://cherne.net/brian/resources/jquery.hoverIntent.html):
   For the mouse-over-drop-down menus.

 * jQuery Cycle Plugin (with Transition Definitions)
   (http://jquery.malsup.com/cycle/):
   For the rotating banners system.

 * Modernizr (http://www.modernizr.com):
   For the progressive enhancement with CSS3 features.


INSTALLATION
------------

 * Download the three JavaScript libraries listed in the Requirements section
   above.

 * Place their .js files in the following directories (using the location of
   this theme as the base directory):
   - hoverintent.js: js/hoverintent (so that the JS file is at
     js/hoverintent/hoverintent.js)

   - jQuery Cycle Plugin: js/cycle (so that the JS file is at js/cycle/cycle.js)

   - Modernizr: js/modernizer (so that the JS file is at
     js/modernizer/modernizer.js)

 * Install the theme as you would normally install a contributed Drupal theme.
   See: https://drupal.org/documentation/install/modules-themes/modules-7 for
   further information.
